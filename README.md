# Java Trial

Playground for common features and libraries. Spans Java 8 through 17.

Features tested:

- [x] Optionals
- [x] Records
- [x] Reflection
- [x] Streams
- [x] Threads

Libraries tested:

- [x] Gson
- [x] Guava
- [x] JUnit
- [x] Mockito
- [x] SL4J

## Getting started

Here are the only two commands you need to know:

```bash
# Run the main application
./gradlew run

# Run the test suite
./gradlew test
```
